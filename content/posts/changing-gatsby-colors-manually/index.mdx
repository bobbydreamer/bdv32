**Note** : As of release `@lekoarts/gatsby-theme-minimal-blog": "^3.0.2"`, theme changes are *Non-Shadowing* meaning you have to make changes in the `node_modules`. Earlier these changes can be shadowed and shadowed files were in `src\@lekoarts\gatsby-plugin-theme-ui\index.js`.

Now you will have make these changes in `node_modules\@lekoarts\gatsby-theme-minimal-blog\src\gatsby-plugin-theme-ui\index.js`.

First lets see the themes and later we shall see the code changes.

### # What CSS key correspond to
```sh 
background: page background
primary: post links / selected text
secondary: tags, header links
heading: headings
toggleIcon: light/dark mode
```

### # Making the Non-Shadowing changes

Filepath : `node_modules\@lekoarts\gatsby-theme-minimal-blog\src\gatsby-plugin-theme-ui\index.js`

```js {8-12,14-18,36}
  colors: {
    primary: tailwind.colors.purple[7],
    secondary: `#5f6c80`,
    toggleIcon: tailwind.colors.gray[8],
    heading: tailwind.colors.black,
    divide: tailwind.colors.gray[4],
    modes: {
      light:{
        background: `#bddccc`, /* Background */        
        primary: `#7400b8`,    /* Alink posted color */
        secondary: `#ee4035`   /* Alink top menu color */
      },      
      dark: {
        background: `#1e172c`,
        primary: `#fad141`,
        secondary: `#2ab7ca`,
        heading: `#ff0080`,
        toggleIcon: `#fdf498`,  
        text: tailwind.colors.gray[4],        
        divide: tailwind.colors.gray[8],
        muted: tailwind.colors.gray[8],
      },
    },
  },
...
  styles: {
    root: {
      color: `text`,
      backgroundColor: `background`,
      margin: 0,
      padding: 0,
      boxSizing: `border-box`,
      textRendering: `optimizeLegibility`,
      WebkitFontSmoothing: `antialiased`,
      MozOsxFontSmoothing: `grayscale`,
      fontWeight: 500,
    },
```
